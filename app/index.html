Setup guide

Show ▾

1. Install Claude Code

Requires Node.js 18+ and an Anthropic account.

npm install -g @anthropic-ai/claude-code

2. Start the local server

Run once from the repo directory — this powers the Run buttons.

npm install
npm start

Then open http://localhost:3000 in your browser.

3. Launch Brave with remote debugging

Regular Brave won't work — it must be launched with this flag.

/Applications/Brave\ Browser.app/Contents/MacOS/Brave\ Browser \
  --remote-debugging-port=9222 \
  --user-data-dir=/tmp/brave-debug

Verify it's running:

curl -s http://127.0.0.1:9222/json/version

4. Add Brave MCP to Claude Code

Add this to ~/.claude/settings.json:

{
  "mcpServers": {
    "chrome-devtools": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-puppeteer"],
      "env": {
        "PUPPETEER_WS_ENDPOINT": "ws://127.0.0.1:9222"
      }
    }
  }
}

5. Log into LinkedIn

In the Brave window you just opened, go to linkedin.com and sign in.
